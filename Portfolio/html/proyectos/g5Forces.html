G5 Forces
  • Descripción: En este proyecto ya contabamos con un diseño creado anteriormente, por lo que lo único que teníamos que hacer es que la web pasara de ser estática a dinámica, es decir que pueda leer los datos de una base de datos o de un XML, en este caso guardabamos la información tanto de parques como de sus respectivas atracciones en un archivo XML. Para conseguir el objetivo, tuvimos que crear archivos PHP que recogen datos de formularios y despues modifican los archivos XML. Además de eso tambien son los que se encargan de mostrar los datos en sus respectivos sitios. Tuvimos que crear un servidor Xampp con el que haciamos que funcionara la conexión para la lectura y modificación de los XML.
  • Conocimientos aplicados: HTML, CSS, JavasCript, XML, XSLT, PHP y Xampp.
  • Fecha: Esta página web la creamos durante un mes, en la tercera evaluación del grado superior de DAM (Abril - Mayo 2024)
  • Colaboradores: En este proyecto estuve trabajando con dos compañeros, aunque no formé parte de la creación del diseño.
