Este proyecto está en proceso, esta información será completada una vez se termine el proyecto.
  • Descripción: El proyecto consiste en el desarrollo de parte de una aplicación multicapa y multiplataforma que permita el registro de usuarios en la base de datos de Odoo, así como su posterior identificación, autenticación y acceso a dicha aplicación. El desarrollo de esta aplicación se enmarcará dentro del proceso de creación y lanzamiento de una empresa de software como primer proyecto de desarrollo realizado por dicha empresa. La planificación y seguimiento del proyecto se realizará con un software de gestión empresarial (ERPCRM).
  • Conocimientos aplicados: Java, JavaFX y comunicación en red mediante sockets, programación con hilos, BD PostgreSQL y Odoo.
  • Fecha: Esta aplicación ha sido desarrollada durante 7 semanas como el reto de la primera evaluación del segundo curso del Grado Superior de DAM. (Septiembre - Octubre 2024)
  • Colaboradores: En este proyecto he trabajado con dos compañeros de clase.
